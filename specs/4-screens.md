## seat-seekers.tsx — 착석 희망자 탐색

**역할**: 곧 내려요 (활성 여정)  
**배경**: `#1B1D22`

### 레이아웃

```
[ TopBar ] — 뒤로가기(좌) + 필터 아이콘(우)
[ "하차 예정자" 제목 (16px/600, #E4E5E6) ]
[ "열차 진행 방향" 정렬 기준 (13px/400, #797E86) ]
─────────────────────────────────────────
[ CarTabs (탑승칸 0~9, 선택된 칸 파란색) ]
─────────────────────────────────────────
[ 하차 가까운 순 / 정렬 드롭다운 ]
[ PersonCard 목록 (ScrollView) ]
  - PersonCard × N (접힘/펼침 토글)
─────────────────────────────────────────
[ 바텀 버튼 영역 (393×98, #1B1D22) ]
  [ 열람 중지 버튼 (secondary) | 기록 보기 버튼 ]
```

### PersonCard 상세

접힌 상태 (173×높이):
```
┌─────────────────────────────────────┐
│ [N정거장 후 하차]            [탑승칸 #] │
│ ──────────────────────────────────  │
│ [좌석 구역 위치]                      │
│ ──────────────────────────────────  │
│ [인상착의 첫 줄]                      │
└─────────────────────────────────────┘
```

펼친 상태 (322×높이): 인상착의 전문 + 좌석 상세 이미지

### 정렬 기준

- 하차 가까운 순 (기본)
- 내 탑승칸과 가까운 순
