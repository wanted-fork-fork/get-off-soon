# 구현 계획: 곧 내려요 지하철 좌석 공유 서비스

## 개요

이 문서는 "곧 내려요" 앱을 화면 단위로 단계별 구현하는 계획을 제시합니다. 각 단계는 독립적으로 테스트 가능하며, 점진적으로 기능을 추가해 나가는 방식으로 구성되었습니다.
